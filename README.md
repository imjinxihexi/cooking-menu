## 椰子木瓜清远鸡（两人份）

   > 清远鸡半只 （切块）</br>
   > 椰子一个（椰子水加水做汤底，椰肉切块）</br>
   > 木瓜半个 （削皮、去核、切快）</br>
   > 姜片
   
   + `椰子水` 加入`姜片`
   + 开大火
   + 煮沸 `椰子水`
   + 转小火
   + 加 `椰肉`、`木瓜`   &nbsp;[煮15分钟]
   + 加 `鸡肉`         &nbsp;[煮5分钟]
   + 加 `盐`         &nbsp;[煮1分钟]
   + 捞起，Enjoy It
   
<br>

## 可乐鸡翅

   > 鸡翅根 6 个 </br>
   > 小可乐 250ml </br>
   > 姜片 </br>
   > 蒜 </br>
   > 油 </br>

  + 鸡翅清洗干净
  + 蒜切成蓉
  + 冷水下锅，加入 `鸡翅` `姜片`
  + 等待水沸腾，水倒掉，清洗干净鸡翅
  + 蒜切粒，切一小块姜片
  + 开火，待锅热了加入 `油` `蒜蓉`
  + 待`蒜蓉`变黄色加入`鸡翅`
  + `鸡翅`煎到两面金黄
  + 加入 200ml `可乐`，盖上盖子焖10分钟
  + 打开盖子把`可乐`煮至粘稠状即可
  + 装盘


## 番茄鸡蛋面
   > 番茄 1 个 </br>
   > 鸡蛋 1 个
   > 米粉

   + `番茄`切小块，热油倒`番茄`煸炒，有出沙的感觉
   + 打入`鸡蛋`，加一点`酱油`调味，炒到快熟了
   + 加入提前煮开的`水`
   + 煮到沸腾加`米粉`
   + `米粉`煮到散开就好了（不能煮太久）
